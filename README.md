# Elden Ring Munchkin

Прототип ККИ по мотивам Munchkin с визуальным направлением Elden Ring и раскладкой стола в духе Hearthstone.

Реализовано:
* две колоды: события и награды;
* стартовая рука из обеих колод и добор по клику;
* перетаскивание врагов на поле боя;
* перетаскивание предметов в слоты инвентаря;
* предпросмотр сыгранных карт через модальное окно.

Используемые технологии:
* React
* Redux
* dnd-kit
* Typescript
* FSD
* Vite

## Drag and Drop

Проект переведён с React DnD HTML5 Backend на dnd-kit: HTML5 Backend плохо контролировал preview перетаскиваемой карты и курсор, из-за чего браузер мог показывать системный no-drop поверх кастомного интерфейса.

Планируемая доработка:
1. Создание ИИ
2. Добавление анимаций (бой, перетасовка колоды)
3. Полная логика заклинаний и боевых модификаторов
4. Возможность забрать карты из инвентаря
5. Разделение UI на противника/игрока

## Термины проекта

Используем эти термины в интерфейсе, правилах, логах и новых именах:

* Открытие события, не "Вышиби дверь"
* Событие / колода событий, не "Дверь" / "колода дверей"
* Награда / колода наград, не "Сокровище" / "колода сокровищ"
* Бегство, не "Смывка"
* Эффект поражения, не "Непотребство"
* Враг, не "Монстр"
* Предмет, не "Шмотка"
* Боевые бонусы, не "Бонусы"
* Проклятие или негативный эффект для curse-like эффектов

## Запуск

```bash
npm install
npm run dev
```

## Тестовая раздача

Для ручной проверки можно включить предсказуемый порядок колод:

```env
VITE_USE_TEST_DECK=true
```

Без этого флага игра использует обычную случайную перетасовку.
